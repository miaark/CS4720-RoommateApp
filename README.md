# Roomeo

A roommate-finding app for people relocating across the United States.

## Overview
Users create profiles with lifestyle details (cleanliness, sleep schedule, occupation, etc.), 
select a city, and get discovered by others looking for roommates in that area. Searchers can 
filter profiles by city and roommate preferences to find compatible matches, view full profile 
details, and direct message users they're interested in.

## Current Working Screens
- Sign Up (includes Google oAuth)
- Login (includes Google oAuth)
